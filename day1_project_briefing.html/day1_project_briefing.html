What You Actually Do

Detection engineer

Model evaluator

Improvement scientist

Test whether augmentation or cGAN data helps.

P

Product thinker

Make the final checkout useful and explainable.

Where Innovation Fits

Every team can improve one part of the pipeline

D1 - Data and catalog

Better splits, label audits, richer prices, categories, local currency.

D2 - Detection

More realistic scenes, better YOLO settings, harder backgrounds.

D3 - Recognition

Confidence thresholds, nearest neighbors, stronger heads.

D4 - Augmentation

Classical augmentation baseline; optional cGAN or equivalent generation.

D5 - Explainability

Show similar catalog images and Grad-CAM evidence.

Final app

Editable receipt, unknown items, confidence warnings, discounts.

Important Boundaries

Be honest about what the demo is and is not

  • It is a proof-of-concept smart-cart assistant.
  • It is not barcode-level SKU recognition.
  • Prices are editable demo catalog data.
  • Validation metrics matter more than pretty outputs.

Success by Day 5

A user uploads a grocery image and gets an annotated image plus receipt-style output.

The system is useful when it can explain mistakes and estimate uncertainty, not only when it looks impressive.
